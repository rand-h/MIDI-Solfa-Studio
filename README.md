# MIDI-Solfa-Studio
MIDI Solfa Studio avec le moteur de [MIDIano](URL_DE_MIDIANO_ICI) 

## Description
Ce projet a pour but de visualiser et lire les notes d'une partition solfa directement à partir d'un fichier PDF. 

Actuellement, le projet intègre déjà les partitions solfa du recueil **Tiona FFPM**.
